A personally developed flight search app.

Tournament King

A website for creating and managing tournament brackets.

Tournament King was a group project designed to give users a place to create brackets for their tournaments and competitions, share stats, gain rankings and allow others to view match and tournament progress live.

Designed and developed using React/Redux, Javascript, NodeJS, PostgreSQL, Auth0 and Websockets.

Unsplash.com - Clone

Functional clone of Unsplash.com website and UI.

This was a project to better understand dynamic data rendering and UI in a social media context. It features the use of both Unsplash's API for constantly updating live data and it's own NodeJS/SQL server back end to store users and their data.

Developed with React/Redux, Javascript, NodeJS, PostgreSQL, HTML/CSS and Auth0.
